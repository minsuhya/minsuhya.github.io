# 머지 금지 vs 패스트트랙 안전 참조 카드

> 출력해서 코드 리뷰 중에 보이는 곳에 두세요.  
> AI 지원 개발에서 살아남는 팀과 그렇지 못한 팀의 차이는 이 목록이 반사적 반응이 되었는가입니다.

---

## 🔴 머지 금지 — 즉시 에스컬레이션이 필요한 10가지 패턴

예외 없음. 아래 중 하나라도 있으면 더 읽지 말고 시니어 리뷰어에게 즉시 에스컬레이션하세요.

| # | 패턴 | 왜 위험한가 |
|---|------|-----------|
| 1 | 단일 함수에서 **순환 복잡도 > 20** | AI는 기술적으로 작동하지만 유지보수와 테스트가 불가능한 깊이 중첩된 로직을 생성함 |
| 2 | **빈 catch 블록** 또는 예외 삼키기 | 프로덕션에서 묵시적 실패; 오류가 추적 없이 사라짐 |
| 3 | **하드코딩된 자격증명**, 연결 문자열, 환경별 값 | 공개 레포에 머지되거나 배포되면 즉각적인 보안 사고 |
| 4 | 30줄 이상에서 **제네릭 변수명** (`data`, `result`, `temp`, `obj`) | AI 네이밍은 가독성 시한폭탄; 새벽 2시에 아무도 디버깅할 수 없음 |
| 5 | 외부 API 응답에 **null 체크 없음** | AI는 해피 패스를 가정; 프로덕션 API는 정기적으로 예상치 못한 null을 반환함 |
| 6 | **루프 안에서 동기 호출** (N+1 패턴) | AI는 주저 없이 이것을 생성; 테스트는 통과하고 부하에서 프로덕션이 무너짐 |
| 7 | 공개 진입점에 **입력 검증 없음** | 가장 흔한 AI 보안 허점; 검증이 클라이언트 사이드에만 있는 경우 많음 |
| 8 | **deprecated 라이브러리 사용** (AI가 오래된 문서로 학습) | 오늘은 작동하지만 다음 의존성 업데이트에서 조용히 깨짐 |
| 9 | **과도한 추상화** — 10줄짜리 유틸리티에 인터페이스 3개 레이어 | AI는 방어적 과잉 설계를 좋아함; 의도와 구현이 보이지 않게 분리됨 |
| 10 | **테스트 없음**, 또는 `True == True`만 검증하는 테스트 | AI는 아무것도 의미 있게 테스트하지 않으면서 통과하는 테스트를 작성함 |

## ✅ 패스트트랙 안전 — 10가지 녹색 신호

아래 10가지가 모두 있으면 PR은 리뷰를 빠르게 통과할 수 있습니다.

| # | 신호 | 무엇을 말해주는가 |
|---|------|----------------|
| 1 | **200줄 미만 PR** | 한 번의 집중 세션으로 리뷰 가능; 결함 밀도는 PR 크기와 상관관계가 있음 |
| 2 | **단일 책임** — 하나의 논리적 변경, 하나의 도메인 | 숨겨진 커플링 없음; 롤백이 깔끔함 |
| 3 | **기존 테스트 커버리지 업데이트** | 작성자가 AI가 아닌 직접 변경을 검증함 |
| 4 | **새로운 의존성 추가 없음** | 공급망 리스크 도입 없음 |
| 5 | **보안 관련 코드 미접촉** | 인증, 결제, PII, 세션이 범위에 없음 |
| 6 | **LLM 사전 리뷰어 클린 반환** (레이어 2 통과) | 의미론적 분석에서 리스크 신호 없음 |
| 7 | **도메인 오너 리뷰 완료** | 코드베이스의 이 부분을 소유한 사람이 확인함 |
| 8 | **모든 린터 검사 통과** (레이어 1 통과) | 알려진 취약점 패턴 없음, 시크릿 누출 없음 |
| 9 | **티켓 수용 기준 검증 완료** | PR이 티켓에서 요청한 것을 수행함 — 가정이 아닌 확인 |
| 10 | **롤백 경로가 명확** | 이것이 프로덕션을 깨트리면 되돌리기가 한 명령으로 부작용 없이 가능 |

## 이 카드 활용 방법

- **리뷰어**: 코드를 읽기 전에 처음 60초 안에 머지 금지 목록을 머릿속으로 점검하세요. 10가지 패턴 중 하나라도 발견하면, 더 읽지 말고 즉시 에스컬레이션하세요.
- **작성자**: PR을 열기 전에 패스트트랙 안전 목록을 확인하세요. 10가지 중 7개 미만이면 PR이 리뷰 준비가 됐는지 재고하세요.
- **팀 리드**: 동일한 머지 금지 패턴이 계속 나타난다면, 그것은 수동으로 계속 잡는 것이 아니라 레이어 1에 타겟 자동화 규칙을 추가해야 한다는 신호입니다.
